# 博客开张

欢迎来到我的博客。

这个站点现在改成了通过 Markdown 文件管理文章，你只需要把文章放进 `content/posts` 目录，构建时就会自动生成列表页和详情页。

## 当前结构

- `content/posts/frontend/hello-world.md`
- `content/posts/frontend/nextjs-features.md`
- `content/posts/css/tailwind-setup.md`

## 为什么这样做

1. 内容和代码彻底分离。
2. Git 管理更清晰，文章本身就是版本化文件。
3. 新增文章不需要再改 TypeScript 数据。
