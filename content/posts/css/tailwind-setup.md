# Tailwind CSS 上手思路

Tailwind 的关键不是记住所有类名，而是先建立一套稳定的设计约束。

## 建议

- 先定义颜色、间距、圆角等 token。
- 页面组件优先复用，不要让类名无序增长。
- 配合 `@tailwindcss/typography` 渲染文章正文。
