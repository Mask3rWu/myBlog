# My Blog

一个使用 Next.js 14 (App Router) 和 Tailwind CSS 构建的静态博客。

## 特性

- ⚡ Next.js 14 + React 18
- 🎨 Tailwind CSS
- 📝 TypeScript
- 📱 响应式设计
- 🚀 GitHub Pages 静态部署

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建静态文件
npm run build
```

## 部署到 GitHub Pages

1. 将代码推送到 GitHub 仓库
2. 在仓库设置中启用 GitHub Pages，选择 `gh-pages` 分支
3. 推送代码后会自动部署

> 注意：博客将部署在 `https://username.github.io/myBlog/` 路径下
